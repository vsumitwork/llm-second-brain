# Keyword Density Targets

The specific frequency and placement rules for keywords in [[wildnet-edge]] content. Defined in [[wildnatedge-writing-standards]]. Natural readability always overrides density — these are targets, not hard counts.

## Primary Keyword

- **Frequency:** 6–10 uses per 1,000 words
- **Density range:** 0.8–1.2% (ideal); max 1.5%
- For a 1,500-word blog: 12–18 uses of the primary keyword
- For a 2,000-word blog: 16–24 uses

## Secondary Keywords

- **Frequency:** 2–4 uses each per piece
- Include LSI (Latent Semantic Indexing) terms and semantic variations naturally

## Semantic Keywords

- Natural inclusion only — never force
- These are topically related terms that signal depth to Google's systems

## [[keyword-placement-priority]] (Mandatory Locations)

Primary keyword must appear in:
1. Title / H1
2. First 100 words of body copy
3. At least one H2 heading
4. Meta title
5. Meta description
6. URL slug
7. Conclusion / CTA section

Where appropriate: image alt text, FAQ questions/answers

## Scale Adjustment

For pieces longer than 1,000 words, adjust proportionally:
- 1,500 words → 9–18 primary keyword uses
- 2,000 words → 12–24 primary keyword uses
- Pillar posts (3,000–5,000 words) → 24–50 primary keyword uses (stay ≤ 1.5%)
