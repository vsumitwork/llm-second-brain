## Definition

Keyword placement priority defines where focus and secondary keywords are placed across a tips blog to maximize SEO signal without keyword stuffing. Placement follows a fixed distribution schema tied to the blog's structure.

## Focus Keyword Placement (10-tip blog)

Target: 5–7 uses total. Fixed positions:

| Position | Use |
|---|---|
| H1 / Title | 1 |
| Key Takeaways bullet 1 | 1 |
| Hook paragraph | 1 |
| Tip 2 or 3 body | 1 |
| Tip 6 or 7 body | 1 |
| Conclusion sentence | 1 |

Total: 6 uses. Stays within the 5–7 window without forcing repetition.

## Secondary Keyword Rules

- One use per secondary keyword minimum
- Distribute across tips — never cluster all secondary keywords in the first 3 tips
- Apply keyword grammar rules: add prepositions/articles so keywords read naturally (see [[wildnatedge-writing-standards]])
